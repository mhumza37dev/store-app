Setup Instructions
- Clone repository: `git clone https://github.com/mhumza37dev/store-app.git`
- Prerequisites: Node.js v16+ and npm (or yarn).
- Install dependencies: `npm install`
- Environment: ensure a `.env` file exists with `VITE_BASE_URL=https://dummyjson.com` (the project includes a `.env` with this value).

Steps to run the project locally
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Preview production build: `npm run preview`

Formatting
- If you make changes and want to format the codebase, run the repository's formatter script:

	`./format-all.sh`
